# LSPatch-Noroot-modle

LSPatch 免 Root 模块合集（无需 Root，通过 LSPatch 注入）

## 模块列表（11个）

| # | 模块 | 功能 |
|---|------|------|
| 1 | ShizukuSceneFix | 修复 Scene 在 Shizuku 授权列表不显示 |
| 2 | GameUnlockerPro_NoRoot | 游戏性能（帧率/机型伪装/进程优化） |
| 3 | MicroXEnhancer | 微信/QQ增强（防撤回/自动红包/步数） |
| 4 | PrivacyGuard_NoRoot | 隐私保护（设备ID/位置/传感器伪造） |
| 5 | AdBlockerX_NoRoot | 广告拦截（WebView/OkHttp/黑名单） |
| 6 | BatteryOptimizer_NoRoot | 省电优化（WakeLock/Alarm/Job） |
| 7 | StepModifier_NoRoot | 步数修改（传感器/上报伪造） |
| 8 | NotifyMaster_NoRoot | 通知管理（过滤/防撤回/历史） |
| 9 | AudioBoost_NoRoot | 音量增强（突破上限/低音/均衡器） |
| 10 | VipUnlocker_NoRoot | VIP解锁（Hook会员状态查询） |
| 11 | VideoSaver_NoRoot | 短视频去水印（抖音/快手/小红书） |

## 使用方式

1. 安装 LSPatch（免Root）
2. 下载模块 APK
3. 通过 LSPatch 将模块注入目标应用
4. 重启目标应用生效
